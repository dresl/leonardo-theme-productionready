
=========================
Leonardo Productionready Theme
=========================

Theme for productionready

Installation
============

.. code-block:: bash

    pip install leonardo-theme-productionready

After installation don't forget run ``sync_all`` command.

.. code-block:: python

    python manage.py sync_all
